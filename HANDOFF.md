# IP Watch KZ — памятка для продолжения работы

Последнее обновление: 19.07.2026. Последний коммит: `6a5985c`.

## Как восстановиться на другом ноутбуке

1. **Установить**: [Git](https://git-scm.com/downloads), [Python 3.11+](https://python.org),
   [Claude Code](https://claude.com/claude-code).
2. **Склонировать репозиторий**:
   ```
   git clone https://github.com/laurakanzadayeva-cyber/ip-watch-kz.git
   cd ip-watch-kz
   ```
3. **Установить зависимости**:
   ```
   python -m pip install -r requirements.txt
   ```
4. **Секреты (локально нужны только для теста SMTP/AI)** — скопировать шаблон и заполнить:
   ```
   copy config\credentials.template.json config\credentials.json
   ```
   Вписать в `config/credentials.json` блок `smtp` (Gmail + app-пароль),
   при необходимости `gemini` / `openrouter`. Файл в `.gitignore`, в репозиторий не попадает.
   > На боевом сайте (Streamlit Cloud) секреты уже настроены в **Settings → Secrets**
   > и на другой ноут переносить их НЕ нужно — сайт работает независимо.
5. **Запуск локально**: `run.bat` (или `python -m streamlit run app/main.py`).
   При первом входе локально зарегистрируйтесь заново (в dev-режиме код подтверждения
   показывается на экране, если SMTP не настроен).
6. **Продолжить с Claude Code**: открыть папку проекта в Claude Code и дать эту памятку
   (`HANDOFF.md`) — контекст восстановится.

## ⚠️ Незавершённое: выполнить SQL в Supabase (для журнала событий)

Supabase → SQL Editor → выполнить [installer/supabase_activity_events.sql](installer/supabase_activity_events.sql).
